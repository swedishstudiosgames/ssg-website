How to become an indie game developer?

This guide will show you, How to make an indie game with these tools:

  • W3schools: a website that provides online tutorials, references, and exercises for learning web development. Youtube Reddit
  • Godot: a beginner-friendly game engine that lets you create 2D, and/or 3D games.
  • GIMP: a free, Open-source image editor that lets you create, and edit graphics.
  • Visual Studio Code: a lightweight, and versatile code editor that supports many languages, and extensions.
  • Blender: a free, and open source 3D creation suite that lets you model, animate, and render 3D objects.

To make a game with these tools, you will need to follow these general steps:

  1. Download, and install the tools from their official websites.
  2. Create a new project in Godot, Setup the basic settings, and assets.
  3. Use GIMP to create and edit textures, sprites, and UI elements for your game.
  4. Use Visual Studio Code to write, Debug the code for your game logic, events, and interactions.
  5. Use Blender to create both 2D. and/or 3D-Models, Animations, and environments for your game. (Don't forget to save often, Than use this addons/plugins/extentions (LoopTools, and Rigify), and bake everything in the end before rendering it)
  6. Import/export the assets between the tools/softwares, and test your game in Godot.
  7. Publish your game to your target platforms, and audiences.

Let's get started!
